# python-data-pipeline-basics
Projeto desenvolvido ao final do Mês 1 do roadmap anual de Inteligência Artificial. O objetivo é consolidar fundamentos de Python aplicados a dados reais, incluindo leitura de arquivos CSV, validação e limpeza de dados, uso de módulos padrão (csv, json, pathlib) e geração de relatórios básicos, com versionamento no GitHub.
